# Quality Estimation Classifiers

This directory contains code and models for training quality estimation classifiers, which are used to assess the quality of text data. These classifiers can be employed in various natural language processing tasks, such as grammar error correction (GEC) and text generation quality assessment.

You can try them online here: https://huggingface.co/spaces/lapa-llm/quality-estimation

Links to models: https://huggingface.co/collections/lapa-llm/lapa-v012-pretraining
